# Data Quality Verification Script

A Python script that validates CSV and JSON datasets for missing values,
required fields, and format errors, then exports detailed quality reports.

## Features
- Validates CSV and JSON files against required schema fields
- Detects missing values, invalid email formats, and incorrect data types
- Exports validation results to CSV reports
- Prints summary of total, valid, and invalid records

## Technologies Used
- Python
- CSV
- JSON

## How to Run
1. Clone the repository
2. Add your CSV or JSON file to the folder
3. Run the command: python data_quality.py
4. Check the generated report files for results
